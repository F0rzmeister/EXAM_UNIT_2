# TDD Assignment

This project implements various tasks using Test-Driven Development (TDD) in JavaScript.

## Features

- Math functions
- Array flattening
- Tree node operations
- Book data processing

## Setup

1. Clone this repository:
   ```bash
   git clone <your-repo-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run tests:
   ```bash
   npm test
   ```
